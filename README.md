# Taller de Punteros en C++

Este repositorio contiene ejercicios básicos de punteros en C++.

## Ejercicio 1: Intercambio de valores usando punteros
Se realiza el intercambio de dos números enteros usando punteros, modificando directamente los valores originales.

## Ejercicio 2: Suma de elementos de un arreglo dinámico
Se crea un arreglo dinámico con punteros, se llena con datos del usuario y se calcula la suma de sus elementos.

## Ejercicio 3: Encontrar el mayor elemento
Se implementa una función que recibe un arreglo y retorna el valor máximo usando punteros, sin usar índices.

## Ejercicio 4: Aplicar operación a un arreglo
Se recorre un arreglo y se aplica una operación a cada elemento, como duplicar o triplicar.

## Ejercicio 5: Matriz dinámica con punteros
Se crea una matriz dinámica con doble puntero, se llena con datos del usuario y se calcula la suma de cada fila.
